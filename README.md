## Game Description
A roguelike action RPG where you play as a magical forest courier — race against the clock, fight through enchanted wilderness, and slowly unravel the secrets of the world you call home.

This is one level of the game, featuring two enemy types: Bat and Wendigo.
The time-based special mechanic is not yet fully implemented.

## Controls
The game is keyboard-controlled.

| Action | Key |
|---|---|
| Move | WASD / Arrow Keys |
| Run | Hold Shift |
| Attack | Z |
| Block | Hold X |
| Special Action (animation only) | C |
| Sprint Attack | Shift + Z |
| Quit Game | Esc |

## How to Run
Open the `.exe` file to launch the game.
